# Decision Intelligence Platform

A Business Intelligence Dashboard built using Python, PostgreSQL, Streamlit and Plotly.

## Features

- Interactive Dashboard
- KPI Cards
- Revenue Analytics
- Product Analytics
- Customer Analytics
- Inventory Monitoring
- Monthly Revenue Trends
- AI Business Recommendations

## Tech Stack

- Python
- PostgreSQL
- Streamlit
- Plotly
- Faker
- Pandas
- psycopg2
